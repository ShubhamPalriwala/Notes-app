# Notes App #

This is a Javascript program which can be accessed through the Command Line.

| Command       | Function           | Required  |
| ------------- |:-------------:| -----:|
| add      | Adds a new note | -title -body |
| remove      | Removes a note      | -title |
| list | Lists all the notes and their title |  |
|read | Reads a specific note | -title |

Libraries namely: Chalk, Validator and Yargs were used.

Basics of Javascript was covered in this app.
